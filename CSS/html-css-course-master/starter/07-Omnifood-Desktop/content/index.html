How it works

건강한 식단을 위한 간단한 3단계

01

무엇을 좋아하는지 말해주세요!

무엇을 먹을지 고민하며 시간을 낭비하지마세요! Omnifood AI가 당신의 입맛에 딱 맞는 주간 식단을 제공합니다. 한끼 식사에 필요한 영양소와 비타민, 맛까지 모두 챙겨드립니다!

02

식단이 어땠는지 말해주세요!

매주 Omnifood AI가 당신의 식단을 관리합니다. 원하는 재료로 교환하고 식단을 선택해보세요! 당신의 레시피도 추가할 수 있습니다.

03

음식을 받을 시간을 정해주세요!

최고의 쉐프들이 매일 당신이 선택한 음식을 요리합니다. 음식은 조리 후 당신에게 필요한 시간에 배달됩니다. 매일 다른 시간과 다른 장소로 바꿀 수 있습니다.
